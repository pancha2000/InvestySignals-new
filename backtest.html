🔬 Backtest Engine

Replay historical Binance data through the exact same indicator logic the live dashboard uses — and see how many trades would have been wins, losses, or timeouts. Admin only.

⚠️ What this tests: The technical confluence engine (RSI, MACD, ADX, EMA, Order Blocks, FVGs, Fibonacci). Open Interest and live News data are NOT included — no reliable historical source exists for those. Numbers represent the code's technical core, not a perfect replay of what the AI would have said on any given day. Always combine with judgment.

Configuration

Full History — runs the simulation across all available candles (shows what the system would have done historically).  |  Walk-Forward — splits history into in-sample/out-of-sample windows to check if results hold on unseen data (the honest validation test).
